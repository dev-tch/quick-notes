## command to generate component 

```
ng g c component-name
```

demo : 

cd to folder where your work exist
 
cd /home/tch2025/lbc-front/src/app/lbc-pages/public-rfp


ng g c compositions/rfp-suppliers --project=lbc-pages


## command to generate model

```
ng g cl class-model-name  --type=model

```

demo: 

npx ng g cl models/supplierData --type=model --project=lbc-pages


--skip-tests : to skip test 

## command to generate service 

```
ng g s class-service-name
```

demo: 

ng g s LocationTranslation
